Question 3

You own a license plate manufacturing company. Write a program that takes a population and determines the simplest pattern that will produce enough unique plates. Since all the plates that match the pattern will be generated, find the pattern that produces the least excess plates. Use a combination of letters (A-Z) and numbers (0-9).

Answer

    I created a Java Applet mainly to use a different language, even though applets are a bit outdated. You can input the population and it will return the Letters, Numbers, Remainder and an Example Plate.

    Assumption: At first I had the simplest pattern the pattern with the least amount of characters, but changed it to least amount of characters give or take 1 extra character. Example 260000 using 4 Letters you get 430,976 excess, but using 1 letter and 4 numbers you get 0 excess.
